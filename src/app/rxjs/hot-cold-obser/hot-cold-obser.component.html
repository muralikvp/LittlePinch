hot-cold-obser works!

  • Cold Observabled start to emit values only when we subscribe to them. Hot ones emit always descpite we have subscribers or not
  • cold observables are unicase, and hpst observables are multicase(share value between multiple subscribers)
  • For hot observables Data Source is created and activated outside of observable. For cold ones Data Source will be Inside the observables
